## Описание

Movie Library — простое приложение для личной коллекции фильмов с поддержкой добавления, фильтрации, сохранения и загрузки информации в формате JSON.

Возможности:
- Добавление фильмов с названием, жанром, годом выпуска и рейтингом
- Фильтрация по жанру и году
- Валидация полей (год — целое число, рейтинг — 0..10)
- Хранение фильмов в файле `movies.json`
- Графический интерфейс на Tkinter

## Как запустить

1. Установите Python 3.
2. Скачайте файлы `main.py` и `movies.json` (можно пустой) в одну папку.
3. Запустите приложение:  
   ```bash
   python main.py
